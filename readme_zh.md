# Unicode字符表
这个Java小程序是unicode字符查看和输入器。

直观简化查找字符对应的unicode码时的工作。

# 安装

## Java 环境(JRE)

安装JRE1.7或1.8版本。(Java 8 即为 1.8 ) 

## 下载Release

按照屏幕分辨率和JRE环境下载Jar文件。
